安装配置Git 1.安装Git sudo yum install git 2.配置Git （1）配置用户名和邮箱： git config --global user.name "Liujiayi2" git config --global user.email "example@example.com" git config --list （2）获取ssh密钥： ssh-keygen -t rsa -C "example@example.com" cat /root/.ssh/id_rsa.pub （3）成功建立和Github的联系： ssh git@github.com （4）关联到远程仓库： git remote add origin git@github.com:Liujiayi2/linux_dazuoye.git git push -u origin "main" Ps:如果git push -u origin "main"运行不成功，可以试一下代码：git remote set-url origin git@github.
